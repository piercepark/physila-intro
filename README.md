# 김수현 · 수현쌤 — 프로필 소개 페이지

물리치료사 / 도수치료사 김수현(수현쌤, [@physila_sh](https://www.instagram.com/physila_sh)) 소개 페이지.

정적 HTML 한 페이지로 구성되어 있습니다.

## 파일 구성

```
physila-intro/
├── index.html           프로필 소개 페이지
└── assets/
    ├── full-body.png    포트레이트 이미지
    └── profile.jpg      원형 프로필 이미지
```

## 로컬 미리보기

```bash
python3 -m http.server 8000
# http://localhost:8000 접속
```
